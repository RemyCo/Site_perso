Stage en développement informatique d’un site web en Python
Nixit Informatique, Fribourg, Suisse
Septembre 2018 - Novembre 2018
Au sein d'une entreprise basée à Fribourg, Suisse, j'ai pu, pendant un stage de 3 mois, développer une version deux d'une interface Web en utilisant le FrameWork Flask (Python), mais aussi Bootstrap, l'HTML, le CSS, JavaScript et l'Ajax, le JSON et le SQL. J'ai pu utiliser les outils employés par l'entreprise : Git et Docker, comme des librairies OpenCV et Babel. Mes objectifs étaient les suivants : * Remodeler la partie Administration du site * Ajout de fonctionnalités à la partie Administration * Faire du traitement d'image en fonction d'image envoyée sur un serveur (Pour ajout de Logo automatiquement sur les images) * Créer un TimeLapse à partir d'images sur un serveur * Développer une API pour permettre de ne pas saturer la plateforme Web de la création de TimeLapse * Amélioration de l'UI de la partie Utilisateur comme de la partie Administrateur. * Traduction du code, développé en Anglais, vers du Français. Tous les objectifs ont été développé avant la fin du stage.
